## Usage

Just like [bw](https://bitwarden.com/help/cli/)

## Versions

The versioning follows bitwarden cli release versions with a `v` prefix and
a `-<PATCH>` suffix for fixes concerning this package (starting with `-1`,
naturally sorted).
